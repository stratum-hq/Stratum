## Or use the CLI directly

If you already know what you want, run the create command with a preset string:

```bash
npx @stratum-hq/create my-app --preset postgres-rls-prisma-express
```

The preset format is `{database}-{strategy}-{orm}-{framework}`. See the [quick start guide](/getting-started/quick-start/) for a step-by-step walkthrough.
