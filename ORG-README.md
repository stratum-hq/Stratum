<h2 align="center">Stratum HQ</h2>

<p align="center">
  Hierarchical multi-tenancy for any stack.
</p>

---

**Stratum** is an open-source platform for adding tree-structured multi-tenancy to your application — config inheritance, permission delegation, three isolation strategies, GDPR compliance, and more. Built for MSSP/MSP/client architectures and any product that needs nested tenant boundaries.

### Repositories

| Repo | Description |
|------|-------------|
| [Stratum](https://github.com/stratum-hq/Stratum) | Core monorepo — 8 TypeScript packages, control plane, CLI, React components, docs site |
